# Verti
Example of RWD website
## Tech stack
- Javascript
- HTML5
- SCSS
- RWD

## How to run:
- `git clone git@github.com:karolina-rachuta/Verti_RWD.git`
- `npm install`
- `npm run sass`
- profit!
